# ¬øCu√°ndo usar `var` y `let`?

::: top

- La principal diferencia entre `var` y `let` es el √°mbito de la variable; mientras que `var` tiene un √°mbito de funci√≥n, `let` tiene un √°mbito de bloque, es decir, que la variable declarada con `let` solo existe dentro del bloque en el que fue declarada.

:::
::: left

```js

```

:::
::: right

```js
}

```

:::
::: bottom

- Una raz√≥n de peso para elegir `let` por sobre `var` es prevenir el _hoisting_, que significa que las variables declaradas con `var` son movidas al inicio de la funci√≥n o del script, lo que puede llevar a errores dif√≠ciles de depurar.
